使用rem前动态配置html的fontSize

方法1：使用js 
// 习惯以iPhone6(375px)为基础,设置字体为100px,淘宝使用
var fontSize = document.documentElement.clientWidth / 3.75;
document.doucmentElement.style.fontSize = fontSize + 'px';
方法2：使用vw 
/* 
    vw把屏幕分成100份，屏幕宽度为100vw
    以iPhone6（375px）为例：1vw代表3.75px
    表示100px使用： 100 / 3.75 = 26.666667vw
*/
html{
    font-size: 26.666667vw;
}
为了避免font-size的继承传递，根据需要设置body的字体 
body{
    font-size: 16px;
}
